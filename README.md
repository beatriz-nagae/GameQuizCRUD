# GameQuizCRUD
Sistema de Quiz com CRUD Completo - Spring Boot + PostgreSQL  Aplicação backend desenvolvida em Java 21 com SpringBoot.

Aplicação backend desenvolvida em Java 21 com Spring Boot para gerenciamento 
completo de quizzes educacionais. Inclui operações CRUD para perguntas, 
sistema de criação de quizzes com seleção aleatória, validações robustas e 
tratamento de exceções profissional. Projeto acadêmico da disciplina de 
Ciência da Computação.

Tech Stack: Java 21, Spring Boot, JPA/Hibernate, PostgreSQL, Lombok, Maven

Criada por Beatriz Nagae em novembro de 2025
